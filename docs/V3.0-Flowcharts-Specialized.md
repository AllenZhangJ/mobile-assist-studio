# V3.0 Flowcharts Specialized

本文件是 V3.0 流程图专项产物。它不是提示词，而是可直接渲染的 Mermaid 图表真源。

设计约束：

- 单人、单设备、本地 Mac 桌面工作站。
- 支持 iOS 或 Android 当前设备，但同一时刻只有一台当前设备。
- Target Library 不升为 L1，作为连接设备、录制和编辑流程的共享能力。
- 低置信不盲点，默认暂停或失败收口。
- 任务状态、用例结果和问题类型分离。

## 2. Product Total Flow

```mermaid
flowchart TD
  Open[打开 App] --> Status{能开始吗}
  Status -- 未连接 --> Connect[连接设备]
  Status -- 可继续 --> Continue[继续当前任务]
  Status -- 有问题 --> Fix[定位问题]

  Connect --> Detect[自动识别平台]
  Detect --> Preview[显示设备画面]
  Preview --> Target[创建目标]
  Preview --> Record[开始录制]
  Target --> Flow[编辑流程]
  Record --> Promote[生成流程]
  Promote --> Flow
  Flow --> Check{预检通过}
  Check -- 否 --> Fix
  Check -- 是 --> Confirm[确认运行]
  Confirm --> Run[串行执行]
  Run --> Result{执行结果}
  Result -- 完成 --> Review[查看结果]
  Result -- 暂停 --> Pause[人工介入]
  Result -- 失败 --> Review
  Pause --> Preview
  Review --> FixTarget[修复目标]
  Review --> FixFlow[修复流程]
  FixTarget --> Target
  FixFlow --> Flow
```

## 5. Device Resource Lock State Machine

```mermaid
stateDiagram-v2
  [*] --> idle
  idle --> remoteControl: 用户远控
  idle --> recording: 开始录制
  idle --> targetCapture: 创建目标
  idle --> running: 开始运行
  idle --> diagnosing: 本机检查

  remoteControl --> idle: 动作完成
  recording --> idle: 停止录制
  targetCapture --> idle: 保存或取消
  diagnosing --> idle: 检查完成

  running --> paused: 低置信或人工点
  running --> stopping: 安全停止
  running --> idle: 运行完成
  running --> error: 设备或驱动错误

  paused --> targetCapture: 更新目标
  paused --> idle: 收口暂停
  paused --> stopping: 停止任务

  stopping --> idle: 原子动作结算
  error --> idle: 用户处理后重置
```

## 7. Target Resolver Decision Flow

```mermaid
flowchart TD
  Start[开始解析目标] --> Load[读取目标定义]
  Load --> Platform{平台支持}
  Platform -- 否 --> Unsupported[平台不支持]
  Platform -- 是 --> Source[采集截图和可用源码]

  Source --> Selector{有可用 Selector}
  Selector -- 是 --> TrySelector[尝试 Selector]
  Selector -- 否 --> Visual{有图像或区域}

  TrySelector --> SelectorOk{命中稳定}
  SelectorOk -- 是 --> Resolved[返回位置和证据]
  SelectorOk -- 否 --> Visual

  Visual -- 是 --> TryImage[尝试图像或区域]
  Visual -- 否 --> Text{需要文字}

  TryImage --> ImageOk{置信度足够}
  ImageOk -- 是 --> Resolved
  ImageOk -- 否 --> Text

  Text -- 是 --> TryOCR[尝试 OCR]
  Text -- 否 --> Coord{允许坐标兜底}

  TryOCR --> OCROk{置信度足够}
  OCROk -- 是 --> Resolved
  OCROk -- 否 --> Coord

  Coord -- 是 --> Coordinate[返回坐标兜底]
  Coord -- 否 --> Low[低置信]

  Coordinate --> Evidence[写入解析证据]
  Resolved --> Evidence
  Low --> Policy{节点策略}
  Unsupported --> Policy
  Policy -- 暂停 --> Paused[进入暂停]
  Policy -- 失败 --> Failed[失败收口]
```

## 8. Workflow Execution Flow

```mermaid
flowchart TD
  Preflight[运行前检查] --> Ready{全部通过}
  Ready -- 否 --> Block[禁用开始并定位]
  Ready -- 是 --> Confirm[确认运行]
  Confirm --> Lock[获取资源锁]
  Lock --> Node[读取当前节点]
  Node --> Type{节点类型}

  Type -- 普通动作 --> Action[调用平台动作]
  Type -- 目标动作 --> Resolve[解析目标]
  Type -- 断言 --> Assert[执行断言]
  Type -- 人工点 --> Manual[暂停等待确认]

  Resolve --> Conf{置信度足够}
  Conf -- 是 --> Action
  Conf -- 否 --> Pause[暂停介入]

  Action --> Event[写入事件]
  Assert --> Result[写入用例结果]
  Manual --> Pause
  Event --> Next{还有节点}
  Result --> Next
  Pause --> Finish[收口到可操作状态]
  Next -- 是 --> Node
  Next -- 否 --> Complete[完成运行]
  Complete --> Release[释放资源锁]
  Finish --> Release
```
